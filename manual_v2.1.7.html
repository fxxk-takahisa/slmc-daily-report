システム概要

SLMC日次報告書作成システムは、4つのブロック（東、センター、西、DCL共通）の日次業務データを簡単に管理し、レポートを自動生成するWebアプリケーションです。

主な特徴

マルチブロック対応

4つのブロックを1つのシステムで管理。ブロック別に最適化されたフォーム。

自動集計

月累計・平均値を自動計算。手入力不要で正確なデータ管理。

多様なレポート

日次・週次・月次・累計の4種類のレポートを自動生成。

簡単メール送信

Gmail/Outlook.com対応。複数宛先に一斉送信可能。

一括エクスポート

複数レポートをZIPでまとめてダウンロード。

データ管理

編集・削除・インポート・エクスポート機能完備。

新機能 v2.1.7

v2.1.7の主な新機能

NEW 1. Webメール対応

Gmail、Outlook.comのWeb版から直接メールを送信できるようになりました。

  • Gmailで開くボタン：Gmailのブラウザ版で新規メールを作成
  • Outlook.comで開くボタン：Outlook.comで新規メールを作成
  • メールソフトで開くボタン：従来のmailto:方式（従来通り）
  • クリップボードにコピーボタン：レポートをコピーして手動で貼り付け
メールソフトがインストールされていない環境でも、WebブラウザからGmail/Outlook.comで送信できます。

NEW 2. システム内マニュアル閲覧

ダッシュボードの「マニュアル」ボタンから、システム内で操作マニュアルを閲覧できます。

  • 別ファイルを開く必要なし
  • モーダルウィンドウで表示
  • いつでもすぐに参照可能

NEW 3. 複数レポート一括ZIP出力

日次・週次・月次・累計レポートとCSVを選択して、ZIPファイルで一括ダウンロードできます。

  • チェックボックスで出力項目を選択
  • 1つのZIPファイルにまとめてダウンロード
  • 複数のレポートを効率的に管理

NEW 4. ドラッグ&ドロップインポート

データインポートがさらに簡単になりました。

  • ドラッグ&ドロップ：ファイルをドラッグするだけでインポート
  • ZIP対応：ZIPファイルを直接インポート可能
  • CSV対応：CSVファイルからもインポート可能
  • JSON対応：従来通りJSONファイルもサポート

はじめに

1. システムへのアクセス

ブラウザでシステムのURLを開きます。推奨ブラウザ：Chrome、Edge、Safari、Firefox

2. ブロック選択

ダッシュボードから、報告するブロックを選択します。

  • 「東ブロック」「センターブロック」「西ブロック」「DCL共通」のいずれかを選択
  • 選択したブロックに応じたデータ入力フォームが表示されます
  • ブロックはいつでも「ブロック選択に戻る」ボタンで切り替え可能
ヒント：マニュアルを確認したい場合は、ダッシュボードの「マニュアル」ボタンをクリックしてください。

データ入力

基本的なデータ入力手順

  1. 日付の入力
    報告日を選択します。デフォルトで今日の日付が設定されています。
  2. BASE名・報告者名の入力
    BASE名（例：西新井）と報告者名を入力します。
  3. 各項目の数値入力
    各セクション（架電、純増、VC/ClubEXなど）の数値を入力します。
  4. データ保存
    「データを保存」ボタンをクリックして保存します。

便利機能

前日データコピー

「前日データをコピー」ボタンをクリックすると、前日のデータが自動で入力されます。

  • 日付、BASE名、報告者名以外の数値が自動入力
  • 変更が必要な箇所だけ修正すればOK
  • 作業時間を大幅に短縮

月累計の自動計算

月累計・平均値は自動計算されます。手入力は不要です。

  • 既存データから当月分を自動集計
  • 平均値も自動計算（小数点第1位まで表示）
  • 入力ミスを防ぎ、正確なデータ管理

入力バリデーション

保存時に未入力項目がある場合、自動的にチェックされます。

  • 未入力項目は赤枠でハイライト表示
  • エラーサマリーで一覧表示
  • 最初のエラー項目に自動スクロール
注意：日付とBASE名、報告者名は必須項目です。必ず入力してください。

レポート生成

レポートタブへの移動

画面上部のタブから「レポート」タブをクリックします。

レポートの種類

1. 日次レポート

指定した日付の1日分の報告書を生成します。

  • 日付を選択
  • 「レポート生成」ボタンをクリック
  • レポートが表示されます

2. 週次レポート

指定した週（月曜〜日曜）の合計値を表示します。

  • 基準日を選択（その週の任意の日）
  • 「レポート生成」ボタンをクリック
  • 週の開始日〜終了日の合計が表示されます

3. 月次レポート

指定した月の全データの合計値を表示します。

  • 年と月を選択
  • 「レポート生成」ボタンをクリック
  • その月の合計が表示されます

4. 累計レポート

指定した期間の合計値を表示します。

  • 開始日と終了日を選択
  • 「レポート生成」ボタンをクリック
  • 期間内の合計が表示されます

レポートのダウンロード

テキストダウンロード

「テキストダウンロード」ボタンで、レポートを.txtファイルとして保存できます。

CSVダウンロード

「CSVダウンロード」ボタンで、全データをCSV形式で出力できます。

  • Excel で開ける形式
  • BOM付きで日本語も正しく表示
  • 全ブロックの全データを一括出力

NEW 一括ZIP出力

複数のレポートとCSVをまとめてZIPファイルでダウンロードできます。

  1. 出力項目を選択
    チェックボックスで必要な項目を選択（日次/週次/月次/累計/CSV）
  2. 期間を設定
    各レポートタイプの日付・期間を設定
  3. ZIPダウンロード
    「選択項目をZIPでダウンロード」ボタンをクリック
  4. 確認
    ZIPファイルがダウンロードされ、選択した全ファイルが含まれています
便利：週次報告を複数人に配布する際、ZIPファイル1つで全資料を送付できます。

メール送信

メール送信の基本手順

  1. レポート生成
    まず、送信したいレポートを生成します
  2. メール送信ボタン
    「メール送信」ボタンが表示されるのでクリック
  3. 送信先を選択
    送信先をチェックボックスで選択（複数選択可能）
  4. 送信方法を選択
    以下の4つから選択します

NEW 送信方法の選択

1. Gmailで開く

Gmailのブラウザ版で新規メールを作成します。

  • Gmailにログインしていれば、新しいタブで作成画面が開きます
  • 宛先、件名、本文が自動入力されます
  • メールソフトがなくても送信可能

2. Outlook.comで開く

Outlook.comのブラウザ版で新規メールを作成します。

  • Outlook.comにログインしていれば、新しいタブで作成画面が開きます
  • 宛先、件名、本文が自動入力されます
  • Outlookアプリがなくても送信可能

3. メールソフトで開く

従来通り、パソコンにインストールされているメールソフトで開きます。

  • Outlook、Thunderbird、Mailなどのアプリが起動
  • 宛先、件名、本文が自動入力されます

4. クリップボードにコピー

レポート内容をクリップボードにコピーします。

  • 任意のメールソフトやWebメールに手動で貼り付け
  • 宛先、件名、本文がすべて含まれています
  • どの環境でも確実に使用可能
推奨：Webメール（Gmail/Outlook.com）を使用している場合は、対応するボタンを使用すると便利です。
注意：レポートが非常に長い場合、URL長さ制限により一部が切れる可能性があります。その場合は「クリップボードにコピー」をご利用ください。

複数宛先への一斉送信

チェックボックスで複数の送信先を選択すると、全員に同時送信できます。

  • 送信先は事前に「メール設定」タブで登録
  • 複数選択した宛先数が表示されます
  • セミコロン区切りで自動設定（エラーメッセージなし）

データ管理

データ検索

保存されたデータを日付範囲で検索できます。

  • 開始日と終了日を選択
  • 「検索」ボタンをクリック
  • 該当するデータが一覧表示されます

データ編集

保存済みのデータを後から修正できます。

  1. データ一覧から「編集」ボタンをクリック
  2. 編集モーダルが開きます
  3. 数値フィールドを修正（日付・BASE名・報告者名は変更不可）
  4. 「保存」ボタンで変更を確定

データ削除

不要なデータを削除できます。

  • データ一覧から「削除」ボタンをクリック
  • 確認メッセージが表示されます
  • 「OK」で削除実行
警告：削除したデータは復元できません。削除前に必ずバックアップを取ることをお勧めします。

データエクスポート（バックアップ）

全データをJSONファイルとしてエクスポートできます。

  • 「全データエクスポート」ボタンをクリック
  • JSON形式でダウンロードされます
  • 定期的にバックアップを取ることを推奨

NEW データインポート

バックアップしたデータをインポートできます。

インポート方法

  1. インポートモーダルを開く
    「データインポート」ボタンをクリック
  2. ファイルを選択
    以下のいずれかの方法でファイルを指定：
    • ドラッグ&ドロップ：ファイルをエリアにドラッグ
    • ファイル選択：「ファイルを選択」ボタンをクリック
  3. 対応ファイル形式
    • .json - JSONファイル
    • .csv - CSVファイル
    • .zip - ZIPファイル（JSON/CSVを含む）
  4. プレビュー確認
    インポートするデータが表示されます。重複データも確認できます。
  5. インポートモードを選択
    • 追加モード：既存データを保持して追加（重複はスキップ）
    • 置き換えモード：既存データを削除して置き換え
  6. 実行
    「インポート実行」ボタンをクリック
便利：ZIPファイルを直接ドラッグ&ドロップするだけで、自動的に中身を解析してインポートできます。

メール設定

送信先の登録

  1. 「メール設定」タブをクリック
  2. 送信先名を入力（例：PCD日次報告）
  3. メールアドレスを入力
  4. 「追加」ボタンをクリック

送信先の管理

登録済みの送信先は一覧表示されます。

  • 送信先名とメールアドレスが表示されます
  • 「削除」ボタンで送信先を削除できます
ヒント：よく使う送信先は事前に登録しておくと、レポート送信が簡単になります。

トラブルシューティング

よくある問題と解決方法

Q1. データが保存されない

A. 以下を確認してください：

  • 日付、BASE名、報告者名が入力されているか
  • ブラウザのローカルストレージが有効か
  • プライベートブラウジングモードを使用していないか

Q2. メール送信ボタンが表示されない

A. 先にレポートを生成してください。レポート生成後にボタンが表示されます。

Q3. Gmail/Outlook.comボタンが動作しない

A. 以下を確認してください：

  • Gmail/Outlook.comにログインしているか
  • ポップアップブロッカーが有効になっていないか
  • レポートが長すぎる場合は「クリップボードにコピー」を使用

Q4. 前日データコピーができない

A. 前日のデータが保存されていない可能性があります。データ管理タブで確認してください。

Q5. ZIPファイルのインポートができない

A. 以下を確認してください：

  • ZIP内にJSON/CSVファイルが含まれているか
  • ファイルのデータ形式が正しいか
  • ファイルサイズが大きすぎないか（20MB以下推奨）

Q6. グラフが表示されない

A. データが1件以上保存されているか確認してください。データがない場合、グラフは表示されません。

Q7. 更新したのに変更が反映されない

A. ブラウザのキャッシュをクリアしてください：

  • Windows: Ctrl + Shift + R
  • Mac: Cmd + Shift + R
それでも解決しない場合：ブラウザのコンソール（F12キー）でエラーメッセージを確認し、システム管理者にお問い合わせください。
